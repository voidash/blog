November 21, 2025 · voidash

  • Google defines agency as “one’s ability to act on their will.”. How to become more agentic outlines the following

    • You should face rejection early in your career. Apply for jobs you really don’t think you’ll get.
    • seek real feedback. Here is my feedback form . FYI it’s anonymous
    • increase your surface area of luck. The person says that more you are confident that conversation is likely to be relevant, less the opportunities.
    • assume everything is learnable.
    • learn to set boundaries. you have to set the time when you will not take action on any calls and then set a time when effort is not tolerated. I want to emphasize on the last item on the list. The world benefits from a person being passive. The distraction dystopia created by the internet is designed to snatch way the agency. Boundaries are really important to set. Being mindful about the time you spend is itself a hard thing to do, but you can just start by listing out all the activities you remember and give it a good or a bad score.

  • Git bisect is a powerful tool.

    git bisect start
git bisect bad <commit-hash>
git bisect good <commit-hash/tag>
git run ./run_something.sh
git bisect log # to show what has been done so far

  • The Trustless Manifesto . It separates the ethos of organizations that enforce “privacy by policy” with “privacy by design”, orgs that follow “don’t be evil” with “impossible to be evil.” Trustlessness is not a feature to add after the fact. It is the thing itself. Without it, everything else - efficiency, UX, scalability is a decoration on a fragile core.

    if simplicity comes from trust, it is not simplicity but a surrender.

    I wonder if something similar to “arrow’s impossibility theorem” in democracy applies to trustlessness. I don’t know if true Trustlessness is even possible.
